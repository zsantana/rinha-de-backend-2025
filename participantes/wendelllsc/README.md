# Rinha de Backend 2025 - CCS1201

## Tecnologias utilizadas

- **Java 21**
- **Spring-boot 3.5.3**
- **Postgres**
- **Nginx**
- **Docker**
- **Maven**

## Execução

1. Subir containers dos Processors conforme instrução oficial.
2. Subir container da API:
   ```sh
   docker compose up -d
   ```
